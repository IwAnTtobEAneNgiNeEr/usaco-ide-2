# Keyboard Shortcuts

Press **`?`** anywhere (outside a text field) to see this list in the app.

| Action | Keys |
|--------|------|
| Command palette (find any problem / command) | `Ctrl + K` (also `Ctrl + P`) |
| Run | `Ctrl + Enter` |
| Judge all | `Ctrl + Shift + Enter` |
| Save | `Ctrl + S` |
| New problem | `Ctrl + N` |
| Ask AI Coach (jump to chat) | `Ctrl + ;` |
| Ask AI about the selected code | `Ctrl + Shift + E` |
| Toggle Explorer (left column) | `Ctrl + B` |
| Zen focus mode | `Alt + Z` |
| Close the top-most modal | `Esc` |
| Show this shortcut list | `?` |

On macOS, `Cmd` works in place of `Ctrl`.

## Editor (Tab snippets)

Type a snippet name and press **Tab** to expand it:

`fastio`, `fori`, `forj`, `rep`, `forn`, `pb`, `eb`, `all`, `vi`, `vll`, `vvi`, `pii`,
`pll`, `ll`, `ld`, `sortv`, `mod`, `inf`, `readn`, `yes`, `no`, `main`.

The editor also does auto-pairing of brackets/quotes, auto-indent, and block indent/outdent
on selections with `Tab` / `Shift+Tab`. Browse the full algorithm snippet library with the
**⚡ Snippets** button.

## Command palette tips

`Ctrl+K` searches both your problems and every app action, and it's **diacritic-insensitive** —
typing `cham bai` matches *Chấm bài*, `giai thich` matches *Giải thích lời giải*.
