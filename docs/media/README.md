# Media

Screenshots and the demo GIF referenced by the root `README.md` live here. Capture these
before publishing and drop them in with these exact names:

| File | Suggested shot |
|------|----------------|
| `editor.png` | The three-column layout: problem list, editor with code, Run console showing an **AC** verdict + diff. |
| `lab.png` | The 🧪 Stress Lab mid-run (generator + brute, a found mismatch). |
| `visualizer.png` | The 🔬 Visualizer rendering a graph or grid from a test input. |
| `demo.gif` | 10–15s: paste a screenshot → 🔎 Analyze → write code → Judge → **AC**. |

Tips: use a clean window (Zen mode `Alt+Z` hides side panels), the dark theme, and a
real-looking problem. Keep the GIF under ~5 MB so it loads on GitHub.
